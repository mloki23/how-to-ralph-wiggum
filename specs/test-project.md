# Simple Trading Signal Generator

Build a Python script that generates trading signals.

## Requirements
- Read CSV data (date, open, high, low, close, volume)
- Calculate 14-period RSI
- Generate signals: BUY when RSI < 30, SELL when RSI > 70, HOLD otherwise
- Write signals to output.json

## Acceptance Criteria
- Script runs without errors
- RSI calculation matches standard formula
- Signals are correctly identified
- Output is valid JSON format
